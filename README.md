## 소개
- 임산부와 임산부좌석을 매칭시켜주는 스마트 임산부 좌석 시스템.
- 좌석 매칭뿐만 아니라 좌석난방조절 및 임산부에게 도움이 되는 정보, 사이트맵 제공
- 임산부석은 Arduino로, 매칭 시스템은 안드로이드 Application으로 구현.
